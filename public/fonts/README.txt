فونت Vazirmatn (اختیاری)
=========================

پیام‌های ارسالی از "Send Message" بدون هیچ فایلی هم فارسی را کامل و خوانا
نمایش می‌دهند (با فونت‌های پیش‌فرض ویندوز مثل Tahoma / Segoe UI که پشتیبانی
کامل فارسی دارند). اما اگر می‌خواهید دقیقاً از فونت Vazirmatn استفاده شود،
کافیست همین دو فایل را (رایگان و متن‌باز) اینجا کنار این README قرار دهید:

  public/fonts/Vazirmatn-Regular.woff2
  public/fonts/Vazirmatn-Bold.woff2

منبع رسمی: https://github.com/rastikerdar/vazirmatn (پوشه‌ی fonts/webfonts)

بعد از قرار دادن فایل‌ها نیازی به تغییر کد نیست — سیستم به‌صورت خودکار در
لحظه‌ی ارسال پیام، این دو فایل را می‌خواند و به‌صورت base64 داخل همان پنجره‌ی
پیام (که روی کلاینت باز می‌شود) جاسازی می‌کند، بدون نیاز به اینترنت روی
سیستم کلاینت.
